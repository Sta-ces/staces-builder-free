== Description ==
Staces Builder FREE optimizes website management with simple content modification tools. Easy to manage and upgrade, this theme is ideal for both beginner and advanced users looking for a streamlined, user-friendly WordPress experience.

== Installation ==
1. Upload the `staces-builder` folder to the `/wp-content/themes/` directory, or install directly through the WordPress theme installer.
2. Activate the theme via the 'Themes' menu in WordPress.
3. Go to Customize to adjust settings as needed.

== Frequently Asked Questions ==

= Does Staces Builder FREE support full-site editing? =
Yes, Staces Builder FREE is optimized for full-site editing, allowing users to customize all aspects of their site.

= Is this theme compatible with the latest WordPress version? =
Yes, this theme has been tested with WordPress 6.3.

= What PHP version is required? =
The theme requires PHP 8.0 or higher for optimal performance.
